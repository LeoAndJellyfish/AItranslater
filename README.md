# AItranslater
翻译工具，分为本地与api两种模式。

# 本地模式

## 安装环境
```
pip install streamlit transformers torch openai
```

## 启动
```
streamlit run app.py --server.address 127.0.0.1 --server.port 6006
```

## 使用模型
Helsinki-NLP/opus-mt-zh-en
Helsinki-NLP/opus-mt-en-zh


# API模式
